# AutoVideoDownloader
a video downloader created by bash command
# Environment
 ffmpeg
 wget
 awk
# Useage
 Download the "autovideodownloader.sh" ,open the command line ,and run the "autovideodownloader.sh".
 Follow the prompts to use the script.
 Good Luck :)
